# CHANGELOG.md

## [v2.4.1] - 2023-10-15
### Added
- Enhanced NVIDIA Reflex support for better frame time consistency.
- New option to enable/disable dynamic resolution for various in-game modes (e.g., Casual, Competitive).

### Fixed
- Resolved an issue where frame rate cap settings would not apply correctly in certain CS2 map rotations.
- Fixed a bug causing the app to crash when switching profiles quickly.

### Improved
- Optimized the frame rate smoothing algorithm resulting in more stable frame rates during intense gameplay moments.
- Improved UI responsiveness and loading time when launching the app.

---

## [v2.3.0] - 2023-09-03
### Added
- Introduced custom frame rate profiles tailored for popular community servers.
- Supports automatic updates for leaderboard-specific optimizations for the competitive scene.

### Fixed
- Corrected a memory leak that could occur when running the app for extended periods.
- Fixed the flickering display issue on high-refresh monitors when using V-Sync.

### Improved
- Updated the user interface for a more streamlined experience.
- Enhanced compatibility with new CS2 map packs, ensuring optimal performance settings.

---

## [v2.1.0] - 2023-06-10
### Added
- Preliminary support for AMD FSR 2.0 integration, allowing for upscaling while maintaining high frame rates.
- A new performance logging feature to analyze frame rates during gameplay sessions.

### Fixed
- Patched an issue where the frame limiter would occasionally be reset after CS2 updates.
- Addressed a bug where the app could fail to recognize certain graphic card drivers.

### Improved
- Fine-tuned algorithm for automatic frame rate adjustments based on in-game performance metrics.
- Added tutorials and tips within the app to help new users optimize settings effectively.
